# Classification Rules

## Categories

| Emoji | ID | Label | Definition |
|-------|----|-------|------------|
| 🎯 | core_work | Core Work | Direct output: job applications, resume tailoring, interview prep, portfolio work |
| 🔧 | exploration | Exploration | Work-adjacent but not direct output: planning, reviewing tasks, optimizing systems, learning tools, friction logs |
| 💰 | side_hustle | Side Hustle | Manufactured spending, mileage transfers, credit card research |
| 🏃 | fitness | Fitness | Gym, exercise, sports |
| 👥 | social | Social | Calls/hangouts with girlfriend, family, friends; networking |
| 🍳 | life_admin | Life Admin | Doing a specific task: cooking, eating, walking dog, errands, cleaning, groceries |
| 🎮 | leisure | Leisure | Active entertainment + aimless browsing. When in doubt, choose this |
| 😴 | rest | Rest | Body is recovering: sleep, naps, intentional rest. Holding your phone ≠ Rest |

## Classification Principles

1. **Classify by behavior, not by purpose.** Record the purpose in the description field. Example: spending 2 hours browsing and comparing gifts = Leisure (behavior is browsing), not Social (purpose is relationship). Picking up a pre-selected gift in 10 minutes = Life Admin.

2. **Rest = your body is recovering.** Sleep, naps, intentional breaks with no screen. If you're holding your phone, it's not Rest.

3. **Life Admin = doing a specific task.** Cooking, walking the dog, picking up packages. Aimless browsing or "random phone time" = Leisure, even if it doesn't feel like entertainment.

4. **When in doubt, choose Leisure.** Better to overestimate Leisure than underestimate it. The purpose of the audit is to expose where time goes.

5. **Add 2-3 words of description to every entry.** Post-hoc analysis depends on this. No need for sentences — "food", "🐶", "phone", "applying" are all fine.

6. **Core Work vs Exploration boundary:** If the activity directly moves a job application, interview, or portfolio piece forward → Core Work. If it supports that work but doesn't directly produce output (planning, reviewing, optimizing tools) → Exploration. The two combined = total work investment. The ratio between them is a key metric.

## Analysis Notes

- **Life Admin fixed costs** (eating ~2h + dog ~30m + hygiene = ~2.5h/day) are unavoidable. When analyzing, separate fixed costs from controllable Life Admin using descriptions.
- **Core Work + Exploration** together represent total work investment. Track the "output ratio" (Core Work ÷ total work investment) as a key efficiency metric.
- **Late-night Leisure** (00:00–02:00) should be flagged separately — it often represents suppressed daytime needs and disrupts sleep quality.
